# gtl

![workflow](https://github.com/sh0e1/gtl/workflows/workflow/badge.svg)

## Overview

Google Translate CLI

## Install

```bash
GO111MODULE=off go get github.com/sh0e1/gtl
go install $GOPATH/src/github.com/sh0e1/gtl
```

## Usage

```
gtl --api-key=YOUR_API_KEY Hello
こんにちは
```
